# テクニカル指標ガイド

## 指標一覧とパラメータ

### SMA（単純移動平均）

- **パラメータ:** 期間 N
- **計算:** 直近 N 本の終値の平均
- **デフォルト:** 20（短期）, 50（中期）, 200（長期）
- **用途:** トレンド方向の判定。短期 > 長期 → 上昇トレンド

### EMA（指数移動平均）

- **パラメータ:** 期間 N
- **計算:** 平滑化係数 k = 2/(N+1)、EMA = 終値 × k + 前回EMA × (1-k)
- **初期値:** 最初の N 本の SMA
- **用途:** SMA より直近の価格に敏感。MACD の基礎

### RSI（相対力指数）

- **パラメータ:** 期間 N（デフォルト 14）
- **計算:**
  1. 各期間の値上がり幅 / 値下がり幅を分離
  2. 平均上昇幅 = 直近 N 本の上昇幅の平均（Wilder の平滑化）
  3. 平均下降幅 = 直近 N 本の下降幅の平均
  4. RS = 平均上昇幅 / 平均下降幅
  5. RSI = 100 - (100 / (1 + RS))
- **判断基準:**
  - 70 以上: 買われすぎ（過熱）
  - 30 以下: 売られすぎ
  - 50 付近: 中立

### MACD（移動平均収束拡散法）

- **パラメータ:** 短期 EMA（12）, 長期 EMA（26）, シグナル期間（9）
- **計算:**
  - MACD ライン = EMA(12) - EMA(26)
  - シグナルライン = MACD ラインの EMA(9)
  - ヒストグラム = MACD ライン - シグナルライン
- **判断基準:**
  - MACD がシグナルを上抜け → ゴールデンクロス（買いシグナル）
  - MACD がシグナルを下抜け → デッドクロス（売りシグナル）
  - ヒストグラムの拡大/縮小 → モメンタムの強弱

### ボリンジャーバンド（BB）

- **パラメータ:** 期間 N（20）, σ倍率（2）
- **計算:**
  - ミドルバンド = SMA(N)
  - 標準偏差 = 直近 N 本の終値の標準偏差
  - アッパーバンド = SMA(N) + σ倍率 × 標準偏差
  - ロワーバンド = SMA(N) - σ倍率 × 標準偏差
- **判断基準:**
  - バンド幅が狭い（スクイーズ）→ ブレイクアウトの前兆
  - 価格がアッパーバンド付近 → 買われすぎの可能性
  - 価格がロワーバンド付近 → 売られすぎの可能性

### ATR（Average True Range）

- **パラメータ:** 期間 N（デフォルト 14）
- **計算:**
  1. True Range = max(high - low, |high - 前足の close|, |low - 前足の close|)
  2. ATR = TR の単純移動平均（Wilder 平滑化を使う実装もあるが、本 skill は SMA 版で統一）
- **判断基準:**
  - 価格と同じ単位の絶対値
  - 価格水準に依存するため、銘柄や時期で水準が異なる
  - ATR が拡大 → ボラ拡大、ATR が縮小 → ボラ縮小（レンジ相場の可能性）
- **用途:**
  - ストップロス幅: ATR × 2 を目安にする手法（ボラに応じた動的設定）
  - ポジションサイジング: ATR が大きい時はサイズを縮める

### ROC（Rate of Change）

- **パラメータ:** 期間 N（デフォルト 12）
- **計算:** ROC = (close - close.shift(N)) / close.shift(N) × 100
- **判断基準:**
  - 0 を上下するシンプルな振動指標（%表示）
  - ROC > 0 → 価格は N 期間前より上昇
  - ROC < 0 → 価格は N 期間前より下落
  - 大きな絶対値は「行き過ぎ」を示唆
- **用途:**
  - モメンタムの方向と勢いを直感的に把握
  - RSI と比べて正規化されていないが、変化率としての解釈が直接的

## 指標の組み合わせパターン

### トレンドフォロー型

1. SMA(50) と SMA(200) のゴールデンクロス/デッドクロスでトレンド判定
2. RSI でエントリータイミングを絞る（押し目で RSI < 40 なら買い）
3. MACD ヒストグラムでモメンタム確認

### 逆張り型

1. RSI が 30 以下 or 70 以上で反転の可能性
2. ボリンジャーバンドのバンドウォーク解除を確認
3. MACD ダイバージェンス（価格と MACD の方向が乖離）

### ブレイクアウト型

1. ボリンジャーバンドのスクイーズを検出（バンド幅が過去 N 期間で最小）
2. 価格がアッパー/ロワーバンドを突破
3. 出来高の増加を確認（`volume` フィールド）
